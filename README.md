## 📖 About

**Velocity** is an open-source, cross-platform code editor designed to deliver a **native development experience** without the overhead of Electron-based applications.

Built using **Modern C++23** and **Qt 6**, Velocity focuses on performance, responsiveness, and modularity. The goal is to create a powerful editor that remains lightweight while providing the customization and developer experience expected from modern tools.

Rather than competing by adding unnecessary features, Velocity is built on the philosophy that **speed, simplicity, and extensibility** should come first.

---

# Why Velocity?

Modern editors are incredibly powerful, but many achieve that by embedding an entire web browser inside the application. Velocity takes a different approach.

### Velocity is designed to be:

# Building Velocity

## Requirements

* C++23 compatible compiler
* Qt 6
* CMake
* Git

### Clone the repository

```bash
git clone https://github.com/<your-username>/Velocity.git
cd Velocity
```

### Configure

```bash
cmake -S . -B build
```

### Build

```bash
cmake --build build
```

### Run

```bash
./build/Velocity
```
